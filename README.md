# Customer Churn Prediction

## 📌 Problem Statement
Predict whether a bank customer will leave (churn) or stay using machine learning.

## 📊 Dataset
- Source: Kaggle Bank Customer Churn Dataset
- Features: Credit Score, Geography, Gender, Age, Balance, etc.

## ⚙️ Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Encoding
- Model Building (Logistic Regression, Random Forest)
- Model Evaluation

## 🤖 Models Used
- Logistic Regression
- Random Forest Classifier

## 📈 Results
- Accuracy: ~85% (depends on run)

## 💡 Business Impact
Helps banks identify high-risk customers and take preventive actions.

## 🛠️ Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
